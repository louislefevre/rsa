# RSA Algorithm
Demonstrates how the RSA algorithms work.

## How to Run
To run from the source code:
```
cd src/main/java
javac Main.java
java Main
```

To run from the JAR file:
```
java -jar rsa-algorithm.jar
```

Note that both the compiler and runtime environment require minimum Java 1.8.
